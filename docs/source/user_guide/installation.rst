Установка и обновление
======================

.. toctree::
   :maxdepth: 3
   :hidden:
   :caption: Содержание:

Установка с помощью `PyPi <https://pypi.org/project/openav/>`_
---------------------------------------------------------------

.. code-block:: sh

   pip install openav

Обновление с помощью `PyPi <https://pypi.org/project/openav/>`_
----------------------------------------------------------------

.. code-block:: sh

   pip install --upgrade openav
